Color

Color: Readonly<Color>
  • Defined in Sprite.d.ts:84

You cannot modify the values of this Color class directly. Instead, replace the entire class with a new object.

Example:

const faded = copyColor(sprite.Color);
faded.A = 0.5;
sprite.Color = faded;
