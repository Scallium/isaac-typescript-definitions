DisableMusicLayers

DisableMusicLayers: boolean
  • Defined in mods/MusicModCallback.d.ts:72

True if music layers are disabled in favour of a mod that does not allow them. False by default. Change to true if your mod does not support layers. This is not associated with the MusicManager.DisableLayer() command and takes priority if true.

AddMusicCallback

  • AddMusicCallback(mod: Mod, callback: (id: Music) => undefined | number | LuaMultiReturn<[jingleID: Music, trackID: Music]>, ...tracks: Music[]): void
    • Defined in mods/MusicModCallback.d.ts:19

    Adds a callback to be triggered whenever a track is about to play. Used for changing music.

    Parameters

    • mod: Mod

      The ModReference for your mod (created with RegisterMod())

    • callback: (id: Music) => undefined | number | LuaMultiReturn<[jingleID: Music, trackID: Music]>

      A function that is called every time a track is about to play. The function should return one of the following:

      • a track ID to play that instead
      • a LuaMultiReturn<[jingleID: Music, trackID: Music]> to play a jingle and queue the track
      • 0 to prevent the track from playing, and allow the current one to continue
      • -1 to stop all music
      • undefined to continue to internal code
        • (id: Music): undefined | number | LuaMultiReturn<[jingleID: Music, trackID: Music]>

        • Parameters

          • id: Music

          Returns undefined | number | LuaMultiReturn<[jingleID: Music, trackID: Music]>

    • Rest ...tracks: Music[]

      The tracks that will trigger your function call. If this argument is omitted, all music changes will trigger this callback.

    Returns void

    The ID of the intended boss room entry music for this room (e.g. Satan, Mom). If not in a boss room, it will simply return one of the two generic themes.

GetMusicTrack

  • GetMusicTrack(): LuaMultiReturn<[currentRoomMusicOrJingle: Music, currentRoomMusic?: Music]>
    • Defined in mods/MusicModCallback.d.ts:44

    Returns LuaMultiReturn<[currentRoomMusicOrJingle: Music, currentRoomMusic?: Music]>

    Either the ID of the current room's music, or a LuaMultiReturn containing the current jingle's ID and the current room's track ID.
