AddRoom

  • AddRoom(table: { AdjacentDisplayFlags: undefined | DisplayFlag; AllowRoomOverlap: undefined | boolean; Clear: undefined | boolean; Color: undefined | Color; Descriptor: undefined | RoomDescriptor; DisplayFlags: undefined | DisplayFlag; Hidden: undefined | boolean; ID: string | number; ItemIcons: undefined | number[]; LockedIcons: undefined | string[]; NoUpdate: undefined | boolean; Position: Vector; Shape: RoomShape; Type: undefined | RoomType; Visited: undefined | boolean }): MinimapAPIRoomDescriptor
    • Defined in mods/MinimapAPI.d.ts:60

    Parameters

    • table: { AdjacentDisplayFlags: undefined | DisplayFlag; AllowRoomOverlap: undefined | boolean; Clear: undefined | boolean; Color: undefined | Color; Descriptor: undefined | RoomDescriptor; DisplayFlags: undefined | DisplayFlag; Hidden: undefined | boolean; ID: string | number; ItemIcons: undefined | number[]; LockedIcons: undefined | string[]; NoUpdate: undefined | boolean; Position: Vector; Shape: RoomShape; Type: undefined | RoomType; Visited: undefined | boolean }
      • AdjacentDisplayFlags: undefined | DisplayFlag

        The display flags that this room will take on if seen from an adjacent room. This is usually 0 for secret rooms, 3 for locked rooms and 5 for all others.

      • AllowRoomOverlap: undefined | boolean

        Optional. The API will automatically remove a room if you add this in the same position, setting this to true will disable this functionality.

      • Clear: undefined | boolean

        The clear boolean for the room.

      • Color: undefined | Color

        A Color object that is applied when this room is rendered on the map.

      • Descriptor: undefined | RoomDescriptor

        A RoomDescriptor object if you are attaching a vanilla room to this table. Setting this will cause this room's display flags and clear boolean to be taken from this RoomDescriptor.

      • DisplayFlags: undefined | DisplayFlag

        The display flags for the room. Matches the format of RoomDescriptor.DisplayFlags.

      • Hidden: undefined | boolean

        This room is secret. It will not be revealed by the compass or the treasure map, and it WILL be revealed by the blue map.

      • ID: string | number

        Any value. This is used to identify your room later.

      • ItemIcons: undefined | number[]

        A list of icon IDs that display on the map (e.g. keys and hearts). This will be overridden once the player enters this room.

      • LockedIcons: undefined | string[]

        A list of strings like above, but this is only shown when the player does not know the room's type. (e.g. locked shop, dice room)

      • NoUpdate: undefined | boolean

        A boolean. If true, then this room's minimap appearance will never be changed by the API itself, meaning its display flags, clear status, visited status and pickup icons will have to be updated externally.

      • Position: Vector

        A vector representing the position of the room on the minimap.

      • Shape: RoomShape

        A RoomShape enum value that represents the sprite on the minimap and where icons will be placed.

      • Type: undefined | RoomType

        A RoomType enum value. Optional, but recommended if you want the room to work as expected with minimap revealing items.

      • Visited: undefined | boolean

        Whether the room has been visited or not. This will be set to true as soon as the player's map position is in line with this room.

    Returns MinimapAPIRoomDescriptor
